Troubleshooting server.log

If you don't see server.log in your file manager:

  1. Check the Node.js section in Hostinger panel.
  2. Ensure Application Entry Point is set to server.cjs.
  3. Ensure Application Root is set to the folder where this test-proxy.html file is located.
  4. Click "NPM Install" and then "Restart".
